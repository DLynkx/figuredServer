Welcome

You've stumbled into the home of figured.io. This is an experimental server for launching new and exciting analytics ideas!

This site currently hosts an RStudio server, and in the future may host more ambitious projects. We're building Shiny apps, fixed wing drones, and chasing the dream in the great city of Monrovia! Check back for updates.

Additionally, the RStudio Server instance on this virtual machine offers training in R fundamentals using the Swirl Package. To use the Swirl training package, use the username and password given to you by the site admin, and type library(swirl) to get started. The package will guide you through the rest.
